# ⏱️ Time Machine

![Time Machine Demo](./images/Time_machine.gif)

Простой веб-таймер с управлением через UI и клавиатуру.

[Демо](https://m1rojke.github.io/Time_machine/) • [Код](https://github.com/m1rojke/Time_machine)

## Что внутри

- **HTML5** — семантическая разметка
- **CSS3** — стили с градиентом, grid-layout, анимации
- **Vanilla JS** — класс Timer с управлением отсчётом

## Функции

- Установка времени: 5–55 минут
- Старт/пауза/сброс
- Клавиатура: `↑` `↓` для времени, `Enter` для старта, `Escape` для меню

## Быстрый старт

```bash
git clone https://github.com/m1rojke/Time_machine.git
cd Time_machine
open index.html
```

## Использование

1. Откройте `index.html` в браузере
2. Нажмите `+` или `−` для установки времени
3. Нажмите большую кнопку или `Enter` для старта
4. Нажмите кнопку обновления для перезагрузки
